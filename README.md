# Color Masking using OpenCV

## Overview
This project demonstrates color masking techniques in computer vision using OpenCV.  
The goal is to extract specific objects or regions from an image based on color filtering.

---

## What is Color Masking?
Color masking is a technique used to isolate parts of an image by selecting a specific color range.  
It is widely used in image segmentation and object detection tasks.

---

## Features
- Load and preprocess images
- Convert images to HSV color space
- Apply color range filtering
- Create binary masks
- Extract target objects from images

---

## Techniques Used
- Color space conversion (RGB → HSV)
- Thresholding based on color ranges
- Bitwise operations
- Image masking

---



## Key Observations
- Works best with distinct color backgrounds
- Sensitive to lighting conditions
- HSV color space improves detection accuracy compared to RGB
- Useful for simple segmentation tasks

---

## Output
The model successfully isolates selected color regions and removes unwanted background areas.
